# 파이프라인 통합 명령 프롬프트

## 🎯 공통 목표
- `src/utils/path_sanitizer.py`의 `sanitize_keyword_for_path` 및 관련 헬퍼를 사용해 세 파이프라인의 입출력 파일명 규칙을 통일한다.
