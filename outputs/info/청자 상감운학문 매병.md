# 청자 상감운학문 매병

## 개요
이것은 '청자 상감운학문 매병'에 대한 테스트용 목업 데이터입니다.
실제 API 호출 대신 반환되는 샘플 데이터입니다.

## 역사 및 배경
- **시대**: 고려시대 (12세기)
- **제작 시기**: 약 1150년경
- **역사적 의미**: 고려청자의 전성기를 대표하는 작품

## 주요 특징
- **외형**: 우아한 곡선과 비취색 유약
- **기술**: 상감 기법의 정교함
- **예술성**: 구름과 학 문양의 조화로운 배치

## 추가 정보
- **소장처**: 국립중앙박물관
- **지정**: 국보 제68호
- **특이사항**: 고려청자 중 가장 완성도 높은 작품으로 평가
